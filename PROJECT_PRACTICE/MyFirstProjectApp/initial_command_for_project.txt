=> create one folder src and create one file app.js in it
=> now, need to start app.js file and write one command 
    e.g. console.log("hello welcome to nodejs app")

=> to execute this file first we need to add this app.js into package.json file

    e.g.
      "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
            "start" : "nodemon src/app.js"   <<---- like this 
        },

=> now, run our project using below command 

    # npm start

now, need to create project structure properly 

projectname 
|- src   // server side all data contain in src folder 
    |- routes 
    |- models
    app.js  (main executable file )
|- public    // all assets will contain in public folder 
    |- css
    |- js
    |- images
|- views  // all designing files 
    |- partial   // global part of html pages 
    index.hbs
